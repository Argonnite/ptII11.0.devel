robotics/RangeSensor

Return the range to an intruder

Version:
  • $$Id: RangeSensor.js 292 2015-09-01 14:16:50Z eal $$
Source:
  • robotics/RangeSensor.js, line 23
Inputs:
Name Type Description
currentPosition JSON The current position of the robot that is searching, typically { "x": 0, "y": 0}.
Outputs:
Name Type Description
rangeMeasurement number The range to the intruder.
Parameters:
Name Type Description
noiseSigma number A measurement of how noisy the signal is (?). Default is 2.0.
intruderKey string The key of the intruder in the key/value store.
storeLocation The URL of the key/value store. The type of this parameter is general so that in Ptolemy the value can be a Ptolemy parameter so that multiple instances of the actor can be easily configured.

Methods

(static) fire()

Get the range to the intruder

The intruder position is obtained by getting the JSON from the URL named by storeLocation + "/get?id=" + intruder The range is then adjusted by a noise sample.
